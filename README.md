## 🤖 Dört Uzman Ajan Sizi Bekliyor

Platform, karmaşık analiz ve üretim süreçlerini her biri alanında uzman dört farklı ajana böler:

1.  **Auditor-2.5 (Tespit Uzmanı):** Metindeki YZ parmak izlerini, sentetik yapıları ve tekrarlayan desenleri saniyeler içinde analiz eder. Risk skorunuzu belirler.
2.  **GhostWriter (Üslup Mimarı):** Metni sizin sesinizden çıkmış gibi doğal, akıcı ve tüm YZ dedektörlerini (GPTZero, Originality vb.) aşacak şekilde yeniden kurgular.
3.  **Sentinel (Doğruluk Muhafızı):** Canlı kaynak doğrulaması yapar. Asla uydurma kaynak (hallucination) üretmez; verileri 2026 güncel internet verisiyle çapraz sorgular.
4.  **StyleMaster (Hafıza Merkezi):** Sizin özel tonlarınızı ve marka sesinizi öğrenir. Steve Jobs tarzı mı? Akademik bir dil mi? StyleMaster hatırlar ve her metinde bu standardı korur.

## 🚀 Öne Çıkan Özellikler

-   **Zero-Hallucination Kaynakça:** Sentinel sayesinde tüm referanslar gerçektir.
-   **Gelişmiş Dilbilgisi Motoru:** Sadece kelime değiştirmez, anlamı koruyarak cümle yapılarını optimize eder.
-   **Premium Glassmorphism Arayüzü:** 2026 tasarım trendlerine uygun, göz yormayan, karanlık mod odaklı modern UI.
-   **Kademeli Üyelik Sistemi:** Free, Pro ve Premium planlarla her ihtiyaca uygun limitler.

## 🛠️ Yerel Çalıştırma

**Gereksinimler:** Node.js (v18+)

1.  Depoyu klonlayın ve dizine gidin.
2.  Bağımlılıkları yükleyin:
    ```bash
    npm install
    ```
3.  `.env.local` dosyasını oluşturun ve `VITE_GEMINI_API_KEY` anahtarınızı ekleyin:
    ```env
    VITE_GEMINI_API_KEY=your_api_key_here
    ```
4.  Uygulamayı başlatın:
    ```bash
    npm run dev
    ```

## 🌐 Dağıtım (Deploy)

Bu uygulama Firebase Hosting için optimize edilmiştir:
```bash
firebase deploy
```
